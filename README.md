# ☸ Nimal-Web Management System

දහම් පාසල් කළමනාකරණ පද්ධතියක් (Nimal-Web).

## 📱 Mobile Friendly & Offline Support
This application is designed with a mobile-first approach. It's fully responsive and works as a **Progressive Web App (PWA)**.
- **Offline Access**: Use the site even without internet.
- **Installable**: Add to your phone's home screen or desktop.

## 🚀 How to Host on GitHub (GitHub Pages)
1.  **Create a Repository**: Go to GitHub and create a new repository named `Nimal-Web`.
2.  **Add Remote**: Open your terminal (or GitHub Desktop) in this folder and run:
    ```bash
    git remote add origin https://github.com/YOUR_USERNAME/Nimal-Web.git
    ```
3.  **Push Data**:
    ```bash
    git push -u origin main
    ```
4.  **Enable GitHub Pages**:
    - Go to **Settings** -> **Pages**.
    - Under **Branch**, select `main` and click **Save**.
    - Your site will be live at `https://YOUR_USERNAME.github.io/Nimal-Web/`
